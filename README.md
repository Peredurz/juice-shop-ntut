# Web Security Personal Assignment
This repository contains the personal assignment for the Web Security course. 
The assignment focuses on identifying and mitigating common web security vulnerabilities.

## **This repository is a modified fork of the OWASP Juice Shop project.**
The changes made to this fork are such that the found vulnerabilities are not exploitable anymore.


## How to run
You should have **Docker** installed on your machine.
And then you can run the following command in your terminal:
```bash
docker compose up
```

The program should be running at: http://localhost:3000
